# LlamaChat Docs

This repository contains the public-facing documentation site for LlamaChat.
It is intended for static hosting, such as GitHub Pages.

AI built by AI, it's AIception!

Suggested use:

- host this directory with GitHub Pages, or
- publish the same contents to any other static hosting provider

Current structure:

- `privacy-policy/` — future public privacy policy site files
- `robots.txt` — crawler guidance for the docs site

Current status:

- `privacy-policy/index.html` contains the human-facing docs page
- `privacy-policy/robots.json` contains a machine-readable summary file
- `banner.png` is the shared docs banner asset
- `robots.txt` provides crawler guidance for the hosted site

Product notes:

- LlamaChat is intended to be:
  - fully local inference
  - no cloud processing
  - encrypted on-device storage
  - no ads
  - no in-app downloads after install
